La propiedad Box-Shadow le permite agregar una sombra paralela alrededor de un cuadreo. Funciona igual que la propiedad Text-Shadow. Chrome, Firefox y Safari admiten esta propiedad con las propiedad con las propiedades: -moz - box -shadow y -webkit-box-shadow. Estos no están en la especificación CSS pero usarlos pueden ayudar a este estilo para trabajar en estos navegadores.
