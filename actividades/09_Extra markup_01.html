Cada elemento HTML puede llevar el atributo id. se utiliza para identuificar de forma única ese elemento de otros elementos de la página. su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carácter). Es importante que no hay dos elementos en la misma página tiene el mismo valor para sus atributos ID (de lo contrario, el valor ya no es único)

dar a un elemento una identidad única le permite darle un estilo diferente que cualquier otra instancia del mismo elemento en la página.

Por ejemplo, es posible que desee asignar un párrafo dentro de la página (quizas un párrafo que contiene una cotización de extracción) un estilo diferentes que todos los demás párrafo. En el ejemplo, el párrafo con el atributo ID cuyo valor es es pullquote se hace en mayúsculas usandp CSS.
